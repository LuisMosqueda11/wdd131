Site Name

Mes que un fan (FC Barcelona fan hub)

The name of this site comes from the iconic phrase in Catalan "MÃ©s que un club" (More than a club), adapted for this website that will target all the loyal fans of this great team.

Site Purpose

The site serves as a comprehensive resource for FC Barcelona fans, providing the latest news, match schedules, player profiles, historical achievements, and more. It aims to be the go-to hub for all things related to FC Barcelona.

Scenarios

  • Do you want to know the origins and history that have shaped Barcelona?
  • Do you want to learn about the current roster and the personal profile of each player?
  • Do you want to know the latest news?
  • Do you want to know the current status of Barcelona in its competitions?
  • Do you want to know the upcoming matches?

Color Schema

Primary Color: #A50044 (Maroon) - Used for headings and accents.

Secondary Color: #004C99 (Blue) - Used for background and section dividers.

Accent Color: #FFD700 (Gold) - Used for buttons and highlights.

Text Color: #1A1A1A (Dark Gray) - Used for paragraph text.

Background Color: #D3DADE (Light Gray) - Used for the overall background.

Typography

Primary Font: 'Bebas Neue' - Used for headings.

Secondary Font: 'Poppins' - Used for body text.
